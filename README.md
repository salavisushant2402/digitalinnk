# Digitalinnk Smart Basket

A supermarket checkout app built with React, Redux Toolkit

## Special Offers
| Offer | Rule |
|---|---|
| Apples 10% Off | 10% off all apple bags |
| Soup & Bread Deal | Buy 2 tins of soup → bread half price |
| Milk 3 for 2 | Every 3rd bottle free |

## Tech Stack
- ⚛️ React 18 + Vite
- 🗄️ Redux Toolkit

## Scripts
```bash
npm install       # install deps
npm run dev       # dev server
```
